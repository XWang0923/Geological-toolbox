# Geological-toolbox
Some useful tools in geological research.

Code list:

ISEA.r
Adopted from "dggridR" package (https://github.com/r-barnes/dggridR). Creating ISEA grids and depositing geological samples into grids

ProxyMap_ISEA2R.m
Extracting useful information (paleocoordinates and geochemical signals) to xlsx or csv, which can be processed by ISEA.r

ProxyMap_R2MATLAB.m
Processing data from ISEA gridding

gplates_modern2paleo.R
Reconstructing the paleocoordinates from modern locations through "rgplates" package (https://gplates.github.io/rgplates/)

MOR.r
Extracting divergence boundary distribution in deep time

MOR.m
Calculating the total length and density of MOR at different latitudes
